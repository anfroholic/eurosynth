EuroSynth — the chip's voices, running in your browser

Every sound below is produced by the chip's bit-exact integer reference models (the same Python that the silicon is verified against), executed live in PyScript / Pyodide. There is no audio sample anywhere on this page — each clip is computed sample-by-sample at the chip's true 12 kHz rate when you press play, then handed to Web Audio. Pick a voice, set the controls, hit ▶ Play (or ⬇ WAV to save the clip).

⏳ Loading Python (Pyodide)… first load pulls the runtime from a CDN (a few seconds). The voices below activate when it's ready.

4 Karplus–Strong — plucked string

A noise burst fills a short digital "string" that rings and mellows. Low N = high note; tiny N = drums.

6 Bytebeat — self-playing loops

A counter t run through a one-line formula; the low 8 bits are the waveform. It plays itself.

5 Chaos — noise, growls, alien drones

Three chaotic maths. Logistic goes tone→warble→noise as r_seed rises; Lorenz is a smooth swooping rumble.

3 SID — 3-voice chiptune

Three oscillators (voice 2 plays an octave below for a thick patch), plus ring-mod & hard-sync for metallic / tearing tones.

7 Neural — the morphing bass (the headliner)

A tiny 5→8→8→1 network is the oscillator — a full forward pass per sample, in fixed-point. One knob, morph, melts the tone through sine → saw → square → pulse. It's a bass (caps near 187 Hz). (Renders take a moment — it's running a neural net per sample in Python.)

How this works. The chip plays one voice at a time at a 12 kHz sample rate. Each engine here is the project's golden integer reference model (models/*_ref.py) ported verbatim — all pure int math, no floats — so the audio is identical, sample-for-sample, to what the silicon produces. PyScript runs that Python in WebAssembly; Web Audio resamples the 12 kHz clip up to your sound card. The lo-fi "crunch" is real, not a filter.

Need to host it: any static server works — e.g. python -m http.server from the repo root, then open /web/. It also runs on GitHub Pages unchanged.
